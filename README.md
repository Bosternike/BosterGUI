# BosterGUI

## Maven
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
```xml
<dependency>
    <groupId>com.github.Bosternike</groupId>
    <artifactId>BosterGUI</artifactId>
    <version>1.1.3</version>
    <scope>compile</scope>
</dependency>
```

## Gradle
```groovy
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
```groovy
dependencies {
    implementation 'com.github.Bosternike:BosterGUI:1.1.3'
}
```
